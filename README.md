## Instalation

```
pkg update && pkg upgrade
pkg i imagegick
pkg i inotify-tools
pkg i nano
termux-setup-storage
cd storage
git clone https://github.com/zidan-dev/raisashot
cd raisashot
bash install.sh
bash raisa.sh -a -d -wm
```
